# Security Policy

## Supported Versions

The following versions of `libexecinfo` are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| < 1.1   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to: [email](mailto:faisalmoshiur+libexecinfo@gmail.com)

Please include the following information in your report:
- Type of issue (e.g., buffer overflow, memory leak in signal handler)
- Location of the vulnerability (file and line number)
- A proof-of-concept or clear steps to reproduce the issue
- Potential impact of the vulnerability

We will acknowledge receipt of your report within 48 hours and provide a timeline for a fix within 5 business days. We follow coordinated disclosure and will ask you to wait until a fix is released before making any information public.
